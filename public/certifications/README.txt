Put your certificate images in this folder so they are served at:

  /certifications/<filename>

This project currently expects these files (case-sensitive on some hosts):

  - programming-in-java-nptel.jpg
  - full-stack-web-development-simplilearn.jpg
  - intro-web-dev-ibm.jpg
  - ai-workflow-ibm.jpg

You can use .jpg/.png/.webp, but if you change extensions or names,
update `src/components/Certifications.tsx` accordingly.
